Explain what wget -nc does. Do not put this text file pg42671.txt in Git. Complete the following loop to tabulate the number of times each of the four characters is mentioned using Linux commands.

Solution

Wget is a networking command-line tool that lets you download files and interact with REST APIs. It supports the HTTP , HTTPS , FTP , and FTPS internet protocols. -nc : In certain cases,the local file will be clobbered, or overwritten, upon repeated download. In other cases it will be preserved. .

wget -nc http://www.gutenberg.org/cache/epub/42671/pg42671.txt
for char in Elizabeth Jane Lydia Darcy
do
  echo $char:
  grep -o $char pg42671.txt | wc -l
done

  1. What’s the difference between the following two commands?
echo 'hello, world' > test1.txt

and

echo 'hello, world' >> test2.txt

Solution

echo 'hello, world' > test1.txt overwrites ‘hello, world’ from command to test1.txt.

echo 'hello, world' >> test2.txt appends ‘hello, world’ from command to test2.txt.
